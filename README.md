# BigBrain Prompting Website

This folder is a static GitHub Pages site for BigBrain Prompting.

## Files

- `index.html`: Marketing page / optional App Store marketing URL.
- `privacy.html`: Privacy policy URL for App Store Connect.
- `support.html`: Support URL for App Store Connect.
- `assets/app-icon.png`: App icon.

## Before Publishing

1. Push this folder's contents to the `Shailey27/BigBrain` repository.
2. In GitHub, enable Pages from the repository root.
3. Use the published `privacy.html` and `support.html` URLs in App Store Connect.

If you publish from `/docs`, copy these files into a `docs/` folder in your website repo.
