## Description Draft

BigBrain Prompting helps you think through what you want before you ask AI.

Use structured task templates to turn rough intent into copy-ready prompts for complex builds, deep research, comparisons, and decision memos. Save reusable prompts to a private local library, favorite the ones you use often, fill `{{variables}}`, and export clean Markdown when you are ready.

Features:

- Task-based prompt templates
- Swipable grouped editor pages
- Prompt readiness checks
- Local saved prompt library
- Favorites, search, categories, and sorting
- Reusable `{{variable}}` fields
- Copy and Markdown export
- No account, API key, or AI provider connection required

BigBrain Prompting does not send prompts to OpenAI, Anthropic, or any other AI model. You choose where to paste or use the final prompt.
